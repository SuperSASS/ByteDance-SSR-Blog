# 静态资源目录

此目录用于存放项目的静态资源文件。

## 用途

- **favicon.ico**: 网站图标
- **robots.txt**: 搜索引擎爬虫规则
- **sitemap.xml**: 网站地图
- **images/**: 静态图片资源
- **fonts/**: 字体文件
- 其他不需要经过构建处理的静态文件

## 访问方式

这些文件会通过以下方式提供服务：

### 开发环境
- Vite 开发服务器会自动提供这些文件
- 访问路径：`http://localhost:5173/文件名`

### 生产环境（SSR）
- Server 通过 `koa-static` 中间件提供这些文件
- 访问路径：`http://localhost:3000/文件名`

## 注意事项

1. 此目录下的文件会被原样复制到构建输出目录
2. 文件名不会被哈希处理，因此更新文件时需要注意缓存问题
3. 对于需要版本控制的资源，建议放在 `src/assets/` 目录下，让 Vite 处理
4. 大型静态资源建议使用 CDN 托管
